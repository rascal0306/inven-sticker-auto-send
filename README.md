# 🎁 인벤 스티커 자동 선물기 

인벤(Inven) 게시판에서 다른 유저에게 스티커를 빠르고 간편하게 선물할 수 있도록 도와주는 크롬 확장 프로그램(Chrome Extension)입니다.  
LLM(Large Language Model)을 통해 생성된 코드를 기반으로 제작되었습니다.

## ✨ 주요 기능

*   **선물 버튼 자동 생성**: 게시판의 닉네임 옆에 선물 상자 아이콘(🎁)이 자동으로 생성됩니다.
*   **원클릭 선물**: 아이콘을 클릭하면 설정해둔 스티커를 자동으로 구매/선물합니다. ***이 때, 5베니가 소모됩니다.***
*   **백그라운드 처리**: 선물 과정(상점 이동 -> 닉네임 검색 -> 구매)이 보이지 않는 팝업 창에서 자동으로 수행되고 즉시 닫힙니다.
*   **스티커 ID 설정**: 확장 프로그램 팝업 메뉴에서 원하는 스티커 번호(ID)를 변경할 수 있습니다. (기본값: 800번)

## 📦 설치 방법 (개발자 모드)

이 프로그램은 크롬 웹 스토어에 등록되지 않았으므로, **'압축 해제된 확장 프로그램 로드'** 방식으로 설치해야 합니다.

1.  이 저장소(Repository)를 **Clone** 하거나 **ZIP으로 다운로드**하여 압축을 풉니다.
2.  크롬 브라우저 주소창에 `chrome://extensions` 를 입력하여 이동합니다.
3.  우측 상단의 **'개발자 모드(Developer mode)'** 스위치를 켭니다.
4.  좌측 상단에 나타난 **'압축 해제된 확장 프로그램 로드(Load unpacked)'** 버튼을 클릭합니다.
5.  다운로드 받은 폴더(소스 코드가 들어있는 폴더)를 선택합니다.

## 🚀 사용 방법

1.  **초기 설정**:
    *   브라우저 우측 상단 확장 프로그램 목록에서 아이콘을 클릭합니다.
    *   선물하고 싶은 **스티커 ID 번호**를 입력하고 '저장하기'를 누릅니다. (기본값은 800번입니다.)
    *   *팁: 스티커 ID는 인벤 아이마트 상점 URL의 숫자 부분입니다. (예: `.../shop/sticker/800`)*

2.  **게시판 접속**:
    *   인벤 게시판 글 목록이나 댓글을 확인합니다.
    *   닉네임 옆에 생긴 **🎁 아이콘**을 클릭합니다.

3.  **선물 확인**:
    *   *"[닉네임] 님에게 [번호] 스티커를 선물하시겠습니까?"* 라는 확인 창이 뜹니다.
    *   확인을 누르면 자동으로 선물이 진행됩니다.

## ⚠️ 주의사항 (필독)

*   **재화 소모**: 이 프로그램은 실제 인벤 포인트(이니/베니/제니 등)를 사용하여 아이템을 구매합니다. 실수로 클릭하여 원치 않는 재화가 소모되지 않도록 주의하세요.
*   **로그인 필수**: 인벤에 로그인이 되어 있어야 정상 작동합니다.
*   **책임 면책**: 본 프로그램 사용으로 인한 재화 손실이나 계정 불이익에 대해 제작자는 책임을 지지 않습니다. 신중하게 사용해 주세요.

## 🛠 기술 스택

*   JavaScript (ES6+)
*   Chrome Extension Manifest V3

## 📂 파일 구조

*   `manifest.json`: 확장 프로그램 설정 파일
*   `background.js`: 백그라운드 로직 및 윈도우 팝업 제어
*   `content_board.js`: 게시판 화면에 선물 버튼 UI 주입
*   `content_shop.js`: 아이마트 상점 페이지 자동화 스크립트
*   `popup.html/js`: 스티커 번호 설정 UI
